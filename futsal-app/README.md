# ⚽ フットサル ローテーション表

7人制フットサルのローテーション管理アプリ（PWA対応）

## 機能
- **試合中モード**: 現在の試合のメンバー配置をカード表示、左右スワイプで試合切り替え
- **全体表モード**: 全試合のローテーション一覧、名前タップでハイライト
- **集計モード**: メンバーごとの出場回数をバーチャートで表示
- **PWA対応**: ホーム画面に追加してアプリとして使用可能

## メンバー
喜種、上野、川邉、畑中、山本、伊達、武田

## ルール
- 各試合: FP 4名 + GK 1名 + 休み 2名
- 「休み→GK」「GK→休み」の連続禁止
- 伊達・武田は遅刻対応として第1試合を休みに配置
- 第8試合以降は7試合周期でループ
